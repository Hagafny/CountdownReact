This is the API

assignmentsIdDelete

Used by the manager in the manager page

URL

http://www.idchomie.com/api/assignments/{id}

HTTP Method

DELETE

Response Type

inline_response_200_1

Parameters

  • Path: id Integer

    The ID of the assignment that needs to be deleted

assignmentsIdsGet

Can be used to either get assignments of 1 class or multiple. Used when we load assignments at the main page

URL

http://www.idchomie.com/api/assignments/{ids}

HTTP Method

GET

Response Type

FullAssignments

Parameters

  • Path: ids Integer

    the ID or IDs of the classs. Use '&' if more than 1 class. For example '11&16'

assignmentsManagerIdsGet

Can be used to either get assignments of 1 class or multiple. Used in the manager page to load only the relevant information of the assignments that we need

URL

http://www.idchomie.com/api/assignments/manager/{ids}

HTTP Method

GET

Response Type

BasicAssignmentsWithIds

Parameters

  • Path: ids Integer

    the ID or IDs of the classs. Use '&' if more than 1 class. For example '11&16'

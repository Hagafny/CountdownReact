This is the API

coursesBasicIdsGet

This is used in both the client page and manager page. In the client page it is the data that populates the course tab. In the manager page it is the data that is seen in the drop down menu of Courses

URL

http://www.idchomie.com/api/courses/basic/{ids}

HTTP Method

GET

Response Type

List

Parameters

  • Path: ids Integer

    the ID or IDs of the classs. Use '&' if more than 1 class. For example '11&16'

coursesIdDelete

Used by the manager in the manager page

URL

http://www.idchomie.com/api/courses/{id}

HTTP Method

DELETE

Response Type

inline_response_200_1

Parameters

  • Path: id Integer

    The ID of the course that needs to be deleted

BasicAssignment

  • courseId : String
    Parent course's ID
  • title : String
    Title of the assignment
  • endDate : String
    The date where the assignment is due. This can be taken straight from the moodle.
  • homeworkUrl : String
    A direct link to the homework PDF/Excel/Word docuemnt

    the assignment identifier in Moodle. This can be found in the submit page of the assignment
  • information : String
    Extra information regarding the assignment
